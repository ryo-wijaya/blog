### Info

This space runs on Jekyll, Liquid templates, Ruby, Github Pages and Actions, Markdown pages and uses the Chirpy Jekyll Theme.

### Development

- Start a local server

  ```shell
  bundle exec jekyll s
  ```

- Clean and build

  ```shell
  bundle exec jekyll clean
  bundle exec jekyll build
  ```

- View all available jekyll commands
  ```shell
  bundle exec jekyll help
  ```

### Posts

- Create a new page

  ```shell
  bundle excel jekyll page "My New Page"
  ```

- Create a new post

  ```shell
  bundle exec jekyll post "My New Post"
  ```

- Create and publish a new draft

  ```shell
  bundle exec jekyll draft "My new draft"
  bundle exec jekyll publish _drafts/my-new-draft.md
  ```

- Template Front Matter

```yml
  ---
  layout: post
  title: <title>
  description: >-
    <description>
  author: ryo
  date: <auto-generated>
  categories: [<cat1>, <cat2>]
  image: <link/to/img>
  toc: true
  comments: true
  pin: false
  published: false
  ---
```
